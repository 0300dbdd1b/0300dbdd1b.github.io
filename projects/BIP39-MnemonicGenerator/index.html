⚔ BIP39 Mnemonic Generator

Bitcoin / mnemonics / Python

A small Python implementation of a BIP39 mnemonic generator.

It was one of my first attempts at understanding how wallet generation works from the inside.

> from entropy to words, from words to addresses.

story :

I made this because I did not like the idea of trusting a closed-source entropy generation process without understanding what was happening.

I wanted to understand every step myself: entropy, checksum, mnemonic words, seed generation, keys, and addresses.

This Python version was my first implementation. I keep it here because I find it funny to look back at it sometimes and see how much I did not know yet.

what it does :

It walks through the process of generating a mnemonic phrase and deriving Bitcoin-related data from it.

The code is not here because it is perfect. It is here because it marks the beginning of a learning path.

status :

educational, old, and kept mostly as an artifact.

Not meant to secure real funds.
