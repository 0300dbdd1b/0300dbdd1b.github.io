⚔ coreparse

Bitcoin Core datadir parser

coreparse is a C API for reading and parsing a Bitcoin Core datadir.

It reads blockchain data directly from disk and exposes it through my own API, so I can inspect the chain without depending only on Bitcoin Core RPC calls.

> reading the chain from disk, not from a black box.

story :

I started coreparse because I was frustrated with using the Bitcoin RPC interface for everything. It is useful, but it can also be slow or limiting when I want to explore chain data in my own way.

I also wanted to understand how Bitcoin Core stores things on disk: block files, indexes, chainstate data, and the relationship between the raw blk.dat files and LevelDB.

So I wrote my own blk.dat parser, added LevelDB interaction, and started building an API around it. The point was not only to gather data from the chain, but to learn by rebuilding the path myself.

what it does :

It parses Bitcoin Core block files, reads blocks and transactions, interacts with LevelDB data, and gives me a C interface to query and experiment with blockchain data.

status :

experimental and evolving.
