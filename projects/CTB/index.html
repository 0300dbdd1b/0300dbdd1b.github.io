⚔ CTB

C toolbox

CTB is my personal C toolbox.

It is made of reusable stb-style headers, macros, types, allocators, containers, and small utilities that I use across my own C projects.

> small pieces of C I do not want to rewrite again.

story :

CTB started because I kept rewriting the same small C helpers in different projects.

I wanted my own toolbox: types, macros, allocators, containers, and utilities that I understand because I wrote them myself.

It is not meant to be a big framework. It is more like a personal bag of tools that grows when I need something.

what it does :

It keeps common C building blocks in one place so I can reuse them without copying the same code from project to project.

status :

experimental and evolving.
